PHPGoogleVoice

A PHP Google Voice API/library

I made this class in order to allow me to easily use Google Voice from PHP. It uses a DOM manipulation class rather than ReGeX, so it should be pretty fast and stable.

It has support for calling, SMS'ing, retrieving messages, and more, just check out the docs.

Check out the PHPDocs here, and to actually get it use an SVN client to checkout: http://botsfordcr.facebook.joyent.us/svn/PHPGoogleVoice/

If something is broken, please contact me at the details listed here so I can fix it!
